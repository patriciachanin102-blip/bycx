# Codex App

元气循环计划移动端 H5，包含前台测评、后台数据查看、CSV 导出和本地 JSON 数据存储。

## Project Structure

- `frontend/`: H5 页面、后台页面和静态资源
- `backend/`: Node.js 原生 HTTP 服务和 JSON 数据文件
- `server.js`: Railway 启动入口
- `package.json`: Railway / Node 启动配置

## Run

```bash
npm start
```

前台：`http://localhost:4173`

后台：`http://localhost:4173/admin`
